# Google Antigravity Knowledge Bundle

This bundle documents the concepts, APIs, and workflows of [Google Antigravity](https://antigravity.google), Google's agent-first development platform. It serves as a reference for agents and humans working with Antigravity's project model, agent lifecycle, artifact system, and security configuration.

## Concepts

- [Platform Overview](concepts/antigravity-platform.md) — Architecture, views, and key tenets of the Antigravity platform
- [Project Model](concepts/projects.md) — Projects as the core organizational unit for agent scope
- [Agent Lifecycle](concepts/agents.md) — Agent states, security presets, and behavior configuration
- [Artifact Types](concepts/artifacts.md) — Verifiable deliverables agents produce: plans, screenshots, recordings
- [Security Presets](concepts/security-presets.md) — Permission scoping, file access, and command execution controls
